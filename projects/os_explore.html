Operating System Exploration

RISC OS

According to Wikipedia, RISC OS was first released in 1987 as the original operating system for the ARM processor, with both the operating system and the processor being the creation of Acorn Computers.

ARM processors are now ubiquitous, since they are the chipset running in cell phones and other mobile devices. They are increasingly making their way back to the desktop as well, with computers such as the Raspberry Pi and the Pinebook.

Features

  • Since RISC OS was designed for 1987 computers, it is small and resource friendly on modern hardward. It runs fast on a Raspberry Pi 1, and provides a GUI which requires a 3-button mouse.
  • It has a built-in BBC BASIC interpreter, and BASIC can be used to interact with the operating system. BBC BASIC has an inline assembler, allowing users to conveniently write ARM assembly language programs.

    • The built-in access to both BASIC and assembly language make RISC OS a good tool for learning programming and how computers work.
