Introduction to Programming with Scratch 3 Projects

The following six Scratch 3 projects are the exercises in the Scratch: Module 1 from the Raspberry Pi projects website.

Scratch Projects

Rock Band
This project introduces the Scratch 3 environment, including the all important stage and sprite objects, and shows how to change costumes and play sounds in reponse to mouse-click-on-sprite events.
Paint Box

This project creates a paint program. It introduces the Pen extenstion, which is used to program a pencil sprite to draw. Message passing is used to set pen color from color sprites. The pen had been a standard feature of Scratch 2, but now it has been moved into and extension.

Extension appear to be a great way to extend the funtionality of Scratch without making the basic interface overwhelmingly complex. In addition to the Pen, extensions for Music, Video Sensing, Text to Speech, Translate, and several that connect to real world sensors are available.
